# Komposify: IoT-Based Smart Composting System

**Komposify** adalah solusi manajemen limbah organik rumah tangga pintar yang mengintegrasikan teknologi IoT untuk mempercepat proses dekomposisi menjadi "Emas Hijau". Proyek ini dikembangkan untuk program **PKM-KC 2026** oleh mahasiswa **ITPLN Jakarta**

## 🚀 Key Features
- **Real-time Monitoring**: Memantau Suhu ($30^{\circ}C-35^{\circ}C$), Kelembapan ($50\%-60\%$), dan pH ($6.8-7.5$) secara akurat.
- **Automated Control**: Pengendalian otomatis kipas, pompa, dan pengaduk berdasarkan parameter sensor.
- **Sustainable Energy**: Beroperasi mandiri menggunakan Panel Surya 100 Wp dan Baterai LiFePO4.
- **Web Dashboard**: Antarmuka responsif untuk pemantauan jarak jauh melalui protokol IoT.

## 🛠️ Tech Stack
- **Frontend**: React.js, Tailwind CSS (v4), Vite.
- **Hardware**: ESP32, DS18B20 (Suhu), Soil Moisture Sensor, Analog pH Sensor.
- **Connectivity**: MQTT / HTTP REST API (Blynk Integration).
